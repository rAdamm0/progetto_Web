# WebLio #
A website recalling an online library.
It allows you to search for university level books in its own library and to book them after Registration/Login.

## Requirements ##

XAMPP(Apache + MySQL)

## Installation and Running ## 

1. Move to the xampp/lampp htdocs folder: Locate first the xampp/lampp directory, then execute
   ```
   cd htdocs
   ```
   

2. Clone the repository
   ```
    git clone https://github.com/rAdamm0/progetto_Web.git
    cd progetto_Web
   ```

3. Open xampp/lampp and activate the Apache and MySQL services

4. Go to the phpMyAdmin query box following this site
   ```
   http://localhost/phpmyadmin/index.php?route=/server/sql
   ```

5. Paste the contents of the following files in the query box and execute it
   ```
   db/database.sql
   db/insert.sql
   ```

6. Go to the following link and enjoy the site!
   ```
   http://localhost/progetto_Web/php/home.php
   ```
